1. site_project_list.html - 工事管理

項目 内容
ページタイトル 工事管理
概要 登録済みの工事案件を一覧表示するメインページ。iPhone 17 Proサイズ（393×852px）に最適化。パネル/リスト切り替え、ステータスフィルタリング機能あり。
デバイスサイズ iPhone 17 Pro（393×852px）

レイアウト構成

エリア 幅 機能・要件
メインコンテンツ 393px カード/テーブル表示（1カラム縦並び）

機能詳細

機能名 詳細 データ項目
ステータスタブ
フィルタータブ ステータス別にフィルタリング（横スクロールボタン型） すべて / 工事中 / 工事完了 / 出来方完了
※各タブに件数表示
表示切替
パネル/リスト切替 ボタンでカード表示⇔テーブル表示を切り替え -
パネルビュー（カード表示）
工事カード カードクリックで詳細ページへ遷移（1カラム縦並び、画像左・情報右） 工事名 / ステータスバッジ / 工事画像（100×100px） / 工事場所 / 工事種類 / 担当者 / 契約日 / 工事開始日 / 完了予定日 / 進捗率 / 未読メッセージ数 / 状況
リストビュー（テーブル表示）
工事テーブル 行クリックで詳細ページへ遷移 状況 / 工事名 / 工事種類 / 工事場所 / 担当者 / 完了予定日 / 未読数

2. site_project_detail.html - 案件トップ

項目 内容
ページタイトル 案件管理（概要）
概要 工事の概要情報を表示するトップページ。iPhone 17 Proサイズに最適化。各カードをクリックで詳細ページへ遷移。

機能詳細

機能名 詳細 データ項目
工事概要カード
工事基本情報 工事種類/責任者（社内）/状況（ドロップダウン・保存ボタン）/協力業者リスト 工事種類 / 責任者（社内） / 状況 / 協力業者リスト（業者名・担当工種・責任者・電話番号）
予算・実績カード
予算実績サマリー 1カラム縦並びで表示 項目 / 予算 / 実績 / 進捗率（直接工事費・経費・合計） / 進捗バー
工事日誌カード
工事日誌新規作成ボタン 緑グラデーションの目立つボタン（カード上部） -
直近2日間の工事日誌 1カラム縦並びで表示 日付 / 業者 / 工種 / 予算 / 当日実績
スケジュールカード
ガントチャート（簡易版） 作業項目/区分（予定・実績）/3月カレンダー/凡例 作業項目 / 区分（予定・実績） / 3月カレンダー / 凡例

5. site_project_detail_report_create.html - 工事日誌作成（ステップ1）

項目 内容
ページタイトル 案件管理（工事日誌作成）
概要 工事日誌を8ステップで作成する最初のページ。ステップ1：概要の入力。

機能詳細

機能名 詳細 データ項目
ステップインジケーター 8ステップ進行状況表示（丸の中に数値、下に小さな文字） 1.概要（アクティブ） / 2.スキャン / 3.作・高 / 4.機・人 / 5.材料 / 6.計器 / 7.点検・記 / 8.写真
プレビューボタン 横幅いっぱいのボタン（プレビューアイコン付き） -
工事日誌タイトル カード上部に「工事日誌作成」を表示 -
工事日誌ヘッダー 基本情報入力 社内工事名 / 協力会社（セレクト） / 担当者名（セレクト） / 確認者
ステップ1：概要 基本情報入力フォーム 作業日時（日付選択） / 天候（セレクト） / 作業時間（開始・終了時刻） / 気温（最高・最低）
アクションボタン 下書き保存・次へボタン 下書き保存 / 次へ
