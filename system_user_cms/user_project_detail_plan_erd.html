プラン作成システム

既存図をベースに複数の改修プランを作成・管理・比較

システム概要

既存データ参照：既存図データを参照（変更不可）
プラン管理：改修プランの作成・管理
CAD編集：図面・要素の詳細編集
仕上変更：外部・内部仕上の変更管理
設備管理：設備情報の変更管理

主要機能

✓ 既存図をベースにしたプラン作成
✓ 部屋の統合・分割・用途変更
✓ CAD図面・要素の詳細編集
✓ 外部・内部仕上表の作成・変更
✓ 設備情報の位置・仕様変更
✓ 工事区分の明確化（新設・変更・解体・残置）
✓ 仕様マスタとの連携

次のステップ

このプランデータをベースに、解体・新設積算システムで 工事区分に応じた詳細な工事費用を自動計算します。
