1. user_project_list.html - 工事リスト

項目 内容
ページタイトル 工事管理
概要 登録済みの工事案件を一覧表示するメインページ。パネル/リスト切り替え、ステータスフィルタリング機能あり。

レイアウト構成

エリア 幅 機能・要件
左カラム 300px 工事名リスト（リンク一覧）→ クリックで詳細ページへ遷移
中央カラム 可変 メインコンテンツ（カード/テーブル表示）
右カラム 300px お知らせ・通知タブ

機能詳細

機能名 詳細 データ項目
ヘッダー部
新規作成ボタン クリックで新規工事登録ページへ遷移 → user_project_create.html
ステータスタブ
フィルタータブ ステータス別にフィルタリング すべて / 案件登録済み / 工事中 / 工事完了 / 出来方完了
※各タブに件数表示
表示切替
パネル/リスト切替 ボタンでカード表示⇔テーブル表示を切り替え -
パネルビュー（カード表示）
工事カード カードクリックで詳細ページへ遷移 工事名 / ステータスバッジ / 工事画像 / 工事場所 / 工事種類 / 担当者 / 契約日 / 工事開始日 / 完了予定日 / 進捗率 / 未読メッセージ数 / 状況
リストビュー（テーブル表示）
工事テーブル 行クリックで詳細ページへ遷移 状況 / 工事名 / 工事種類 / 工事場所 / 担当者 / 完了予定日 / 未読数
右サイドバー
お知らせ・通知 タブ切り替えで表示内容変更 お知らせリスト / 通知リスト（各未読バッジ付き）

2. user_project_create.html - 新規工事登録

項目 内容
ページタイトル 新規工事登録
概要 新規工事案件を登録するためのフォームページ。7つのセクションに分かれ、スクロールスパイ連動のナビゲーション付き。

セクション構成

No セクション名 主要機能 データ項目
1 工事概要 入力フォーム
基幹システム取り込み		 工事名 *必須 / 工事種類 *必須（セレクト：アスファルト舗装工事/コンクリート舗装工事/補修工事/区画線工事/その他）/ 工事場所 *必須 / 発注者 / 契約金額（円）/ 工期開始日 / 工期終了日 / 責任者（社内）/ 現場代理人 / 備考
2 スケジュール テーブル入力
基幹システム取り込み		 工種 / 担当業者 / 開始予定日 / 終了予定日
※行の追加・削除可能
3 図面登録 ファイルアップロード 画像・PDFファイル（ドラッグ＆ドロップ対応）
4 予算 テーブル入力
基幹システム取り込み		 工種 / 仕様 / 単位 / 数量 / 単価 / 金額（自動計算）
※行の追加・削除可能
5 社内担当者 メンバー選択
権限付与		 登録済みメンバー選択（ドロップダウン）/ 権限（管理者/編集者/閲覧者）
※選択後バッジ表示、個別削除可能
6 外部協力会社 会社選択
工種割当		 登録済み協力会社選択（ドロップダウン）/ 担当工種（セレクト）
※選択後バッジ表示、個別削除可能
7 資料 ファイルアップロード
分類チェック		 ファイル選択 / 資料分類チェックボックス（KY活動記録表/安全施工サイクル/施工計画書/品質管理計画書/道路使用許可/契約書類）

操作機能

機能 詳細
基幹システムから取り込み 工事概要/スケジュール/予算の各セクションに配置。クリックでAPIを呼び出しデータ自動入力（実装時）
スクロールスパイ 左ナビゲーションが現在表示中のセクションに連動してハイライト
登録ボタン 左ナビ下部＋ページ下部固定。クリックで確認ダイアログ→登録処理→一覧ページへ遷移
キャンセルボタン 一覧ページへ遷移

4. user_project_detail_report.html - 工事日誌一覧

項目 内容
ページタイトル 案件管理（工事日誌）
概要 工事日誌を日付降順で一覧表示。1日あたり最大3社の工事日誌を表示。

表示構成

要素 詳細
ヘッダー 「工事日誌一覧」タイトル / 件数表示（白文字・黒背景バッジ）例：全25件中 1-15件表示
工事日誌テーブル 列構成：業者名 / 工種 / 予算 / 当日実績
行構成：
・日付ヘッダー行（例：2026/03/14（土））
・業者行（1社2行構成：工種行＋経費行）
・1日あたり最大3社表示
表示日数：約5日分
操作 各行クリックで工事日誌詳細へ遷移（URLパラメータ：date, company）
ページネーション フッターにページ番号（1〜5）/ 前後ボタン
